# DesignStudio - Преміум Веб-дизайн

Професійна веб-студія, що спеціалізується на створенні сучасних та функціональних веб-сайтів.

## Файли проекту

- `index.html` - Основна структура сайту
- `styles.css` - Стилі та оформлення
- `script.js` - JavaScript функціональність

## Структура проекту

Проект демонструє навички в Git контролі версій та веб-розробці.

## Запуск

Відкрийте `index.html` у браузері, щоб побачити сайт.
